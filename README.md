# Music App
A terminal-based app for storing, updating, deleting and listing music and listens/streams. 

## Description
This app runs in a console. Users are able to add both music and listens to a database by typing info. It features a fully functional menu. 

This project uses Kotlin for the code and JUnit5 for writing tests. IntelliJ was the IDE used.
